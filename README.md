# wsx9172.github.io

## 🐍 贪吃蛇游戏

一个现代化的贪吃蛇游戏实现，使用纯 HTML5、CSS3 和 JavaScript 构建。

### 🎮 功能特性

- ✨ 流畅的游戏体验
- 📊 实时得分和等级显示
- 🎯 递进式难度增加（每吃10个食物速度提升）
- 💾 本地存储最高分记录
- 📱 响应式设计，支持各种设备
- 🎨 现代化的UI界面

### 🕹️ 如何玩

1. 点击"开始游戏"按钮开始游戏
2. 使用键盘方向键（↑↓←→）控制蛇的方向
3. 吃掉红色食物增长身体并增加得分
4. 避免撞到边界或自己的身体
5. 按空格键可以暂停/继续游戏

### 📋 游戏规则

- **得分计算**：每吃一个食物获得当前等级数量的分数
- **等级提升**：每吃10个食物等级提升1级，游戏速度相应增快
- **游戏结束**：当蛇碰撞边界或自己的身体时游戏结束
- **最高分**：游戏会自动保存你的最高分记录

### 🎨 界面说明

- **得分**：当前游戏的总分数
- **最高分**：历史最高分数记录（本地存储）
- **速度**：当前游戏等级，等级越高游戏越快

### 💡 技巧

1. 提前规划蛇的路线，避免陷入死胡同
2. 在游戏速度变快时，保持冷静并预判方向
3. 尝试创造螺旋形路线以获取更多食物
4. 利用暂停功能来思考下一步策略

### 🛠️ 技术栈

- **HTML5** - Canvas API 用于游戏渲染
- **CSS3** - 现代化样式设计和响应式布局
- **JavaScript** - 游戏逻辑和交互处理
- **LocalStorage** - 最高分数据持久化
